# Tech-Stack-Recommender
NLP-based Tech Stack Recommender using TF-IDF &amp; Cosine Similarity, built with Python on Android using Pydroid 3. 🚀

# 📱 Tech Stack Recommender

A Natural Language Processing (NLP) project that recommends the most suitable technology role based on a user's skills using **TF-IDF Vectorization** and **Cosine Similarity**.

This project was built with **Python** and developed entirely on an **Android smartphone using Pydroid 3**.

---

# 🎓 Project Overview

This project analyzes the skills entered by the user, converts them into numerical vectors using **TF-IDF (Term Frequency–Inverse Document Frequency)**, calculates the similarity between the user's skills and predefined technology stacks using **Cosine Similarity**, and recommends the most relevant technology role.

This project was developed as part of my **Artificial Intelligence Internship at DecodeLabs** to strengthen my understanding of **Natural Language Processing (NLP)** and recommendation systems.

---

# 📥 Input

The user enters technical skills.

### Example

```text
Python Machine Learning TensorFlow
```

or

```text

```

---

# 📤 Output

The system recommends the most suitable technology role.

### Example

```text

```
